====== MediaPlayer ======
Created Sunday 28 July 2013

It's a multi-step process

1. Setup
2. Request Focus
3. Playback
4. Listen for focus change
5. Listen for errors

Several gotchas:
1. Check for silent mode
2. Use the current system alarm volume
3. Respect phone calls and music player
4. Return audio focus when done
