===== Creating a Delegate Without Declaring It =====
Use Func:

'''
Func<string,List<string>> haha = (emailList) => {
	string[] from = emailList.Split(',');
	List<string> lFrom = new List<string>(from.Length);
	lFrom.AddRange(from);
	return lFrom;
};
'''
