====== ExtendedStoredProcedures ======
Created Friday 24 January 2014

**Note:** As per [[http://technet.microsoft.com/en-us/library/ms175200(v=sql.105).aspx|this article]], extended stored procedures were deprecated after Sql Server 2008 R2. You should use [[ClrIntegration]] instead.

An **extended stored procedure** is a DLL that can be run by the SQL Server. The stored procedure runs inside of the SQL server. You must have permission to run the stored procedure.

Features of extended stored procedures:

* You can pass parameters
* Can return results
